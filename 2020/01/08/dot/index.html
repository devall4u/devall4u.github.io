내적(Dot product)

공식

  • 내적 공식

$$ \color{blue} { \overrightarrow{a}\cdot\overrightarrow{b}= |a||b|\cos\theta
} $$

$\theta: \ 0\leq\theta\leq\pi$
  • 두 벡터의 내적
    $$
    \color{blue} {
    \vec{a}=<a_{1}, a_{2}>, \vec{b}=<b_{1}, b_{2}> \\
    \vec{a}\cdot\vec{b}=a_{1}b_{1}+a_{2}b_{2}
    }
    $$
  • 벡터의 크기(Norm)

$$ \color{blue} { \vec{a}=<a_{1}, a_{2}>,} \color{blue} {|\vec{a}|=\sqrt{a_{1}^{2} + b_{2}^2}} $$

내적과 각도

내적을 사용해 간단하게 두 벡터의 예각, 직각, 둔각을 확인할 수 있다.

값 내용
a·b > 0 예각
a·b = 0 직각
a·b < 0 둔각

응용

각도기(Protractor)

다음은 벡터의 내적을 응용해 모니터 화면에서 각도를 측정할 수 있는 프로그램을 제작했다.
두 벡터의 각 중 $ 0\leq\theta\leq\pi$ 사이의 각을 표시한다.

특징

  • 화면 위의 슬라이더바를 통해 투명도 조절
  • 화면 Pixel 단위의 좌표 및 반지름 표시
  • 프로그램의 크기 조절
  • 측정을 위한 세 점의 이동

내적, 수학

내적(Dot product)

  • 내적 공식

$$ \color{blue} { \overrightarrow{a}\cdot\overrightarrow{b}= |a||b|\cos\theta} $$
$\theta$는 $0\leq\theta\leq\pi$ 사이의 값이다.

내적을 사용해 간단하게 두 벡터의 예각, 직각, 둔각을 확인할 수 있다.

값 내용
a·b > 0 예각
a·b = 0 직각
a·b < 0 둔각

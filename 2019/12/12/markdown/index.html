Markdown 간단 사용법

개요

Markdown은 텍스트 문서 양식을 편집하는 문법이다. 기능이 복잡하지 않아 배우기 쉽고 문서 작성 작업에 더 집중할 수 있다. 티스토리나 Github 등 다양한 곳에서 많이 사용하고 있다. 장점으로는 HTML로 변환이 쉽게 가능하다는 것이다.
아래 사용법과 결과를 따라해보면 금방 적응이 가능하다.
